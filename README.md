# 自定义脚手架练习

### 相关依赖

-   commander: 命令行处理工具
-   chalk:命令行输出美化工具
-   inquirer:命令行交互工具
-   ora: 终端 loading 美化工具
-   git-clone:下载模板工具
-   figlet:终端生成艺术字
-   fs-extra:用来操作本地目录

### 如何发布

1.注册 npm 账户
2.npm login 注意是 npm 官方源 https://registry.npmjs.org/
3.npm publish 4.包名不能重复，重复了就修改 package.json 的 name
